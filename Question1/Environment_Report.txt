Question 1 - Linux Environment Verification

Objective

Verify the Linux/macOS environment and collect user account, shell, directory, file system, and network information.

---

1. Username Verification

Command:
whoami

Output:
krishikajoshi

Observation:
This command displays the username of the currently logged-in account. The active user is krishikajoshi.

---

2. User Groups Verification

Command:
groups

Output:
staff everyone localaccounts _appserverusr admin _appserveradm _lpadmin _appstore _lpoperator _developer _analyticsusers com.apple.access_ftp com.apple.access_screensharing com.apple.access_ssh com.apple.access_remote_ae com.apple.sharepoint.group.1

Observation:
This command displays all groups associated with the current user account. Group memberships determine the permissions and access rights available to the user.

---

3. Current Shell Verification

Command:
echo $SHELL

Output:
/bin/zsh

Observation:
This command displays the default shell being used by the current user. The system is using the Z Shell (zsh).

---

4. Current Working Directory

Command:
pwd

Output:
/Users/krishikajoshi/Question1

Observation:
The pwd command displays the absolute path of the current working directory. It confirms the location where the assignment files are stored.

---

5. Workspace File and Directory Listing

Command:
ls -la

Output:
total 0
drwxr-xr-x   2 krishikajoshi  staff    64 24 Jun 15:31 .
drwxr-x---+ 38 krishikajoshi  staff  1216 24 Jun 15:36 ..

Observation:
The ls -la command lists all files and directories, including hidden entries, along with permissions, ownership, and other metadata.

---

6. Network Connectivity Verification

Command:
ping -c 4 google.com

Output:
4 packets transmitted, 4 packets received, 0.0% packet loss

Observation:
The ping command verifies network connectivity by sending packets to Google. Successful replies and zero packet loss confirm that the system has internet access.

---

Conclusion

The system environment was successfully verified. User information, shell configuration, working directory, file listing, and network connectivity were checked and found to be functioning correctly.
